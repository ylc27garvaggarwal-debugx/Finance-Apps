# Finance-Apps
A user-friendly finance app that recommends personalized investments based on your amount, risk tolerance, liquidity needs, and time horizon. It features secure login/signup, investment profile management with CRUD operations, and data storage via Supabase, helping you make smarter investment decisions effortlessly.
